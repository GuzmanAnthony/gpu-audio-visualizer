## Project Goal
This project explores GPU-accelerated audio preprocessing and visualization using NVIDIA CUDA and NVIDIA DALI.

We aim to compare:
- CPU audio preprocessing
- Custom CUDA kernels
- NVIDIA DALI GPU pipelines

## Week 1 Objective
- Set up GitHub repository
- Install and test NVIDIA DALI
- Validate GPU audio decoding pipeline

## Project Structure
- dali_pipeline/ – DALI experiments
- cuda_kernels/ – custom CUDA kernels
- visualization/ – rendering
- benchmarks/ – performance comparisons
